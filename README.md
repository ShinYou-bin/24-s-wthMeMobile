## 주기능
- 로그인 기능 : 기본 로그인, 카카오톡 로그인을 할 수 있다.
- 할 일 등록 기능 : 달력에서 날짜를 선택해 할 일을 등록할 수 있다
    - 할 일을 수정, 삭제, 다른 날짜로 이동 시킬 수 있다
- 사용자 팔로우 기능 : 다른 사용자를 팔로우 하여 할 일 여부를 공유할 수 있다
- 팔로우한 사용자의 할 일 확인 기능
